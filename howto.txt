Инструкция к запуску:
1. Запускать модель можно либо через скрипт, либо через web api

2. Для запуска модели на обучение через скрипт, необходимо запустить скрипт go_train.py
Если модель запущена в докере, то
curl -X GET http://localhost:10031/train


Необходимо также существование пути ./saved_model/classify_pt, где будет сохранена модель после тренировки

Обучение производится по полю заголовков title. По полю text я датасет не делал, так как длинные тексты требуют дополнительной обработки перед передачей в модель.
Думаю, что это отдельная тема.
После тренировки автоматически запускается тестирование на тестовом наборе данных.

3. Для запуска модели на предсказание, надо запустить скрипт go_predict.py
Либо curl -X POST http://localhost:80/predict -H "Content-Type: application/json" -d "{\"message\": \"В МИД России отреагировали на сообщения о пленении 37 россиян солдатами ВСУ\"}"
Для предсказания необходимо, чтобы в папке ./saved_model/classify_pt находился файл с обученной моделью saved_model.pt
На гитхабе этого файла нет, так как из-за своего размера он не может быть туда загружен
Этот файл автоматически создается после завершения тренировки модели, которая запускается скриптом go_train.py, как описано в п.1
Скрипт отображает строку ввода, куда пишется сообщение для классификации.

Пример работы программы:

->? В МИД России отреагировали на сообщения о пленении 37 россиян солдатами ВСУ
>>> С вероятностью 84.1% это "Бывший СССР"

->? Голикова заявила о росте смертности из-за COVID-19 в России в октябре
>>> С вероятностью 94.0% это "Россия"
